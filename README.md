# File-Based-Contact-Management-System
**File-Based Contact Management System**  Project Overview The File-Based Contact Management System is a Python application that allows users to manage contacts efficiently using file storage. It performs full CRUD (Create, Read, Update, Delete) operations and stores data persistently in a CSV file.  This project focuses on real-world file handling .
## Features:-
- Add new contacts
- View all saved contacts
- Search contacts by name or phone number
- Update existing contact details
- Delete contacts permanently
- Persistent storage using `.csv` file

---

##  Technologies Used
- Python 3
- CSV file handling
- OS module
